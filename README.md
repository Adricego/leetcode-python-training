# LeetCode Python Training

Repositorio personal para entrenar resolución de problemas de programación usando Python.

El objetivo de este repositorio es prepararme para entrevistas técnicas, fortaleciendo:

- Pensamiento lógico.
- Estructuras de datos.
- Algoritmos.
- Complejidad temporal y espacial.
- Explicación clara de soluciones técnicas.

## Estructura

```txt
problems/
└── 01_arrays_hashing/
    └── two_sum.py
```
